Data transfer and Analysis of Andrea Tropomyosin project
Sylvain Berlemont, 10/09


A. Transfer data from Scripps -> Orchestra

1.$ ssh -X sb234@orchestra.med.harvard.edu
2.$ cd /groups/lccb-interfil/Tropo/Analysis/ROOT_DIRECTORY
3.$ rsync -avz sylvain@lccbfs001.scripps.edu:/data/projects/Tropomyosin/ROOT_DIRECTORY/ .

B. Update FSMCenter & MovieData parameter files (so that I am able to read Andrea's projects)

1.$ cd ~/matlab/projects
2.$ svn update *
3.$ bsub -Is -R "mem > 20000" -q danuser_int_12h matlab -nosplash -nodesktop
4.>> addpath(genpath('/home/sb234/matlab'))
5.>> rootDirectory = '/groups/lccb-interfil/Tropo/ROOT_DIRECTORY'
6.>> fsmUpdateProjDir(rootDirectory)
7.>> relocateMovieData(rootDirectory, false)

C. Run analysis

% e.g. subDirNames = {'Actin', 'TM5NM1'}
% e.g. forceRun = [0 0 0 0 0 0 1]

1.>> batchMakeFigures(rootDirectory, subDirNames, forceRun)
2.>> batchDisplayFigure1(rootDirectory)
3. Save anything needed (figure, etc.)
4. Exit matlab
5. Logout orchestra

D. Transfer analysis from Orchestra -> Scripps

1.$ ssh sylvain@lccbfs001.scripps.edu
2.$ cd /data/projects/Tropomyosin/Analysis/ROOT_DIRECTORY/
3.$ sudo rsync -avz sb234@orchestra.med.harvard.edu:/groups/lccb-interfil/Tropo/ROOT_DIRECTORY/ .
4.$ sudo chown -R abacconi:danuserOld *
5. Logout lccbfs001

E. Update FSMCenter & MovieData parameter files (so that Andrea is able to open his own projects)

1.$ ssh sylvain@lccbsv014.scripps.edu
2.$ cd matlab/projects
3.$ svn update *
4.$ sudo -u abacconi matlabDist -nosplash -nodesktop

8. Exit matlab
