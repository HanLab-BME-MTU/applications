qFSM documentation

Trajectory plot

Plots speckle trajectories.

Method:

1 - Contained in frame range

All the matches in a selected frame range are displayed in one plot. Trajectories longer than the range will be cut.

2 - Originating in frame range

All the trajectories originitating within the selected frame range are displayed in one plot. Trajectories longer than the range will be displayed in their entirety.

Color-code trajectories

Colors (ranging from blue to green) represent time. For method:

1 - Trajectories are multicolor. The color of a segment along the trajectory represents the frame number.

2 - All trajectories starting at a given frame will be colored the same.

The function asks the user to set the frame range of interest via a dialog.

The function returns the applied colorbar in a separate window.
