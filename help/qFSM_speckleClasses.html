Assign speckles to Classes

 

Each speckle from speckleArray is assigned to one of 13 classes:

Class 1: significant BIRTH due to POLYMERIZATION, significant DEATH due to DEPOLYMERIZATION;

Class 2: significant BIRTH due to POLYMERIZATION, significant DEATH due to POLYMERIZATION;

Class 3: significant BIRTH due to DEPOLYMERIZATION, significant DEATH due to DEPOLYMERIZATION;

Class 4: significant BIRTH due to DEPOLYMERIZATION, significant DEATH due to POLYMERIZATION;

Class 5: significant BIRTH due to POLYMERIZATION, non significant DEATH;

Class 6: significant BIRTH due to DEPOLYMERIZATION, non significant DEATH;

Class 7: non significant BIRTH, significant DEATH due to DEPOLYMERIZATION;

Class 8: non significant BIRTH, significant DEATH due to POLYMERIZATION;

Class 9: non significant BIRTH, non significant DEATH;

Class 10: ghost speckle, a subset of class 9;

Class 11: speckle already present in frame 1 which dies within the movie;

Class 12 : speckle born within the movie and still present at movie end;

Class 13 : speckle which lives for the whole movie.

 

The new information is stored into a new structure, speckleClasses, which contains following fields:

.bTime: time of actual birth (not of 'b' speckles, tb+1);

.dTime: time of actual death (not of 'd' speckles, td-1);

.first: index of 'b' speckle in speckleArray;

.first: index of 'd' speckle in speckleArray;

.pos: [y x; y x] := [ Speckle coordinate at actual birth; Speckle coordinate at actual death ];

.class: one of 13 possible classes (see above);

.network: indicates the network (lp, la, or transition) to which the speckle belongs(this field is NOT FILLED in this function).

 

The structure speckleClasses is written to disk as speckleClasses.mat.
