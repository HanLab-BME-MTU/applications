Turnover maps

 

This function creates maps of polymerization, depolymerization and net assembly rate by averaging kinetic scores calculated by SpeckTackle in both time and space.

Time averaging is performed by collecting kinetic scores over n frames. The value for n is set by the user in the "Number of frames for time averaging" field. Kinetic scores are created by SpeckTackle. The user is asked to pick the first kinScore###.mat file in the SpeckTackle project directory. When asked to pick an image, the user can pick any image of the stack, since the image is only used to define the size of the maps to be created.

Space averaging is obtained by a convolution of the time-averaged map with a Gaussian kernel with sigma specifid by the user through the "Sigma for map low-pass filtering (pixels)" field. The user should pick a value for sigma which gives a meaningful kernel support.

The function will either create one (steady-state) map or several, depending on the value of n and the range of frames t the user decides to analyze. In fact, the user is asked to specify the range of frames t to be analyzed through a dialog. Maps will be created over the selected range with a sliding window of the specified number of frames n.

If the number of frames n for time averaging is set to zero, all t frames in the stack will be used to create one single steady-state map. Such a steady state map is displayed in three figures: one polymerization map, one depolymerization map, and an RGB net assembly map, with the polymerization map in the Red channel, and the depolymerization map in the Green channel. These figures are not automatically written to disk. The user can still do this via the figures' menu.

In contrast, if several maps are created over the user-defined range (meaning n<t), the user will be asked to pick a directory where the maps will be automatically saved (as .mat and .tif files).
